# Code of Conduct

Hajimu is a language project for learning, experimentation, and collaboration.
We want the project to be welcoming to beginners, educators, language designers,
and systems programmers.

## Expected Behavior

- Be respectful and patient.
- Assume good intent, especially across language and cultural differences.
- Give feedback on code, behavior, and design, not on personal qualities.
- Explain trade-offs when disagreeing.
- Make room for beginners and first-time contributors.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Sexualized language or attention
- Publishing private information without permission
- Repeated off-topic disruption
- Dismissing people because of language ability, nationality, experience level,
  or background

## Reporting

If you see behavior that makes the project unsafe or unwelcoming, contact the
maintainer privately:

- GitHub: [@ReoShiozawa](https://github.com/ReoShiozawa)

The maintainer may remove comments, close threads, block users, or take other
reasonable action to protect the project community.

## Scope

This code of conduct applies to:

- GitHub issues and pull requests
- documentation and code review discussions
- project-related chat or community spaces
- public communication representing the Hajimu project

## Japanese

相手の経験、母語、国籍、技術レベルを理由に見下す発言や、個人攻撃、ハラスメント、差別的な発言は受け入れません。意見が違う場合も、人格ではなくコード・設計・振る舞いに焦点を当てて議論してください。
